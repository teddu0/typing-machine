# Контекст проекта

## Идея

«Клавишки» — браузерный тренажёр слепой десятипальцевой печати для детей,
которые только начинают знакомиться с компьютером.

## Главные принципы

- Интерфейс должен быть добрым, спокойным и понятным без инструкции.
- Ошибка — это подсказка, а не наказание.
- Новые буквы добавляются постепенно и часто повторяются.
- Крупные элементы, короткие тексты и минимум отвлекающих действий.
- Прогресс сохраняется автоматически.
- Ошибочная английская раскладка не считается ошибкой ученика: интерфейс
  показывает `EN` и помогает переключиться на `RU`.

## Текущий учебный маршрут

Маршрут состоит из пяти последовательных курсов:

1. Средний ряд — 36 занятий.
2. Верхний ряд — 22 занятия.
3. Нижний ряд — 18 занятий.
4. Цифры — 15 занятий.
5. Практика — 24 занятия с постепенным смешиванием рядов, слов и цифр.

Опорные клавиши: `а` и `о`. На них обычно находятся метки для указательных
пальцев. В экранной клавиатуре эти насечки обязательно отображаются.

Курс строится постепенно: отдельная клавиша, соседняя пара, небольшое
смешивание и закрепление. Новые буквы не должны появляться раньше, чем ребёнок
несколько раз повторил уже изученные движения. Текущая версия курса содержит
короткие уровни. Следующий курс открывается после завершения предыдущего.

## Техническая архитектура

- Backend: Node.js без внешних зависимостей.
- Модули Node.js: ES Modules.
- API: `GET /api/courses`.
- Frontend: HTML, CSS и JavaScript.
- Favicon повторяет логотип приложения: фиолетовая клавиша с буквой `А`.
- Браузер не предоставляет системную раскладку напрямую. Интерфейс определяет
  `RU` или `EN` по последней нажатой буквенной клавише.
- Хранение прогресса: `localStorage` браузера.
- Версия учебного курса хранится вместе с прогрессом, чтобы изменения порядка
  уровней не связывали старые звёзды с новыми упражнениями.
- Прогресс хранится по ключу `courseId:levelId`. При переходе на модель курсов
  звёзды старого среднего ряда сохраняются.
- Клиентский флаг `isFullAccess` в `public/app.js` открывает все курсы и занятия
  для ручного тестирования. В обычном режиме он должен быть `false`.
- Буква `Ё` не входит в текущий учебный маршрут и не показывается на экранной
  клавиатуре.
- В хедере доступна отдельная инструкция по постановке рук. Она показывает
  домашний ряд, насечки `А` и `О`, положение больших пальцев над пробелом и
  напоминает держать кисти расслабленными.
- Инструкция содержит интерактивную полную учебную клавиатуру: при наведении
  или фокусе нужный палец подсвечивается и тянется к выбранной клавише, а
  подпись называет палец и руку.
- Руки в инструкции имеют объёмную учебную иллюстрацию с ладонями, запястьями,
  ногтями и суставными линиями. Название активного пальца появляется рядом с
  ним только во время выбора клавиши.
- Контейнеризация: Docker Compose.
