# Клавишки

Дружелюбный тренажёр слепой десятипальцевой печати для детей.

## Запуск

```bash
npm start
```

Затем откройте [http://localhost:3000](http://localhost:3000).

## Запуск в Docker Compose

```bash
docker compose up --build
```

После запуска откройте [http://localhost:3000](http://localhost:3000).
Остановить приложение можно командой:

```bash
docker compose down
```

В первом наборе уровней изучаются буквы среднего ряда русской раскладки:
`ф ы в а п р о л д ж э`.

Упражнения можно выполнять с физической или экранной клавиатуры. Прогресс
сохраняется в браузере автоматически.
